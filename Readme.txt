Descrizione:
Stanco di decidere cosa fare durante la giornata? 
ti vedo già dire "uffi, non so se studiare, guardarmi un film o fare attività fisica" inserisci cosa vuoi fare dentro scelte.txt e lo script farà tutto il resto!

Installazione:
Ti basterà avviare casualità.py ma dovrai installare prima datetime:
windows: py -m pip install datetime
linux: pip3 install datetime
